API Endpoints

POST /inference — whisper.cpp compatible

POST /v1/audio/transcriptions — OpenAI compatible

POST /load — hot-reload model

GET /health — readiness and liveness check

Example (curl)

curl http://127.0.0.1:8080/inference \
  -H "Content-Type: multipart/form-data" \
  -F file="@audio.wav" \
  -F temperature="0.0" \
  -F response_format="json" \
  -F language="en"
